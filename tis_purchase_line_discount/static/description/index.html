Purchase order lines with discounts

This module allows to define a discount per line in the purchase orders. This discount can be also negative, interpreting it as an increment. It also modifies the purchase order report to include the discount field in it.

Configuration

To display purchase order lines with discounts, enable discounts in purchase settings.

Purchase >> Configuration >> Settings
