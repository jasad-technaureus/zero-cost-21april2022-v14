Fiscalisation Form

Fiscalisation Button


After confirming the Invoice, A Fiscalisation Button will be shown.

Successfully Fiscalised


By Clicking the FISCALISE button the Invoice will be fiscalised if the given informations were right. And generate an IIC, IIC Signature, FIC and a QR Code. The XML Request file and the XML Response file also attached there.


If you try to Fiscalise an Invoice which is already fiscalised once, A Message will popup.

The Invoice has not been Fiscalised


If the given informations were wrong the Invoice will not be fiscalised. And an attachment of Error Response will be added.

Add Debit Note


If You create a Debit Note by clicking the Add Debit Note Button, the Corrective IIC Ref and Corrective Issue DateTime automatically filled.


Now it can fiscalise

Add Credit Note (Partial Refund)


If You create a Credit Note (Partial Refund), the Corrective IIC Ref and Corrective Issue DateTime automatically filled.


Now it can fiscalise


Add Credit Note (Full refund and new draft invoice)


If You create a Credit Note (Full refund and new draft invoice), then we can specify an invoice for the reference. Then the Corrective IIC Ref and Corrective Issue DateTime automatically filled.


Now it can fiscalise


Bad Debt


If You Enable the 'Is Bad Debt', You can choose an invoice specifically, Then the Corrective IIC Ref and Corrective Issue DateTime automatically filled. Now it can Fiscalise
